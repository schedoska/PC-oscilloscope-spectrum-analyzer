Detailed Description

Klasa kursora pozwaljącego wykonywać pomiary sygnału określonych chwil czasu.

Kursor jest wyświetlany na obszarze oscyloskopu i może być dowolnie przesuwany wzdłóż osi X, Każde przesunięcie będzie skutkowało oszacowaniem wartości napięcia w danej chwili czasu za pomocą funkcji interpolacji próbki oraz wysłaniem sygnału informującym o aktualizacji danych
